walPutByteArray

open fun walPutByteArray(offset: Long, buf: ByteArray, bufPos: Int, size: Int): Long

Puts instruction into WAL. It should write part of byte[] at given offset. This value returns pointer to WAL, which can be used to retrieve data back with WriteAheadLog#walGetByteArray(long). Pointer is composed of file number, and offset in WAL file.

Parameters

offset - where data will be written in main store, after WAL replay (6 bytes)

buf - byte array of data

bufPos - starting position within byte array

size - number of bytes to take from byte array
Return
